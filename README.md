# 🧮 Python Calculator

A simple command-line calculator built using Python. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- User-friendly input
- Handles invalid operators

## 🛠 Technologies Used

- Python 3

## 📋 How to Run

1. Clone this repository:

```bash
git clone https://github.com/manish-2109/Python-Projects.git
```

2. Navigate to the project folder:

```bash
cd Python-Projects
```

3. Run the program:

```bash
python calculator.py
```

## 💻 Example

```text
Enter first number: 10
Enter operator (+,-,*,/): *
Enter second number: 5

Output: 50
```
